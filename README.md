# 📊 Olist E-Commerce Analytics Dashboard

## Project Overview

This project analyzes sales performance, customer behavior, product performance, and payment trends using the Brazilian E-Commerce Public Dataset by Olist.

The dashboard was built in Microsoft Excel using Power Query, Power Pivot, DAX Measures, Pivot Tables, Pivot Charts, KPI Cards, and Interactive Slicers to provide business insights and support data-driven decision-making.

## Dataset

The project uses the Brazilian E-Commerce Public Dataset by Olist.

Tables used:

- Customers
- Orders
- Order Items
- Order Payments
- Products
- Product Category Translation

---

## Data Model

Relationships were created between the following tables:

- Customers → Orders
- Orders → Order Items
- Orders → Payments
- Order Items → Products
- Products → Product Category

## Key Insights

- São Paulo generated the highest number of customer orders.
- Health & Beauty was the top revenue-generating product category.
- Credit Card was the most preferred payment method.
- Over ₹15.8M revenue was generated from approximately 99K orders.
- More than 96% of orders were successfully delivered.
- Freight costs exceeded ₹2.25M, highlighting opportunities for logistics optimization.

---

## Business Recommendations

- Focus marketing campaigns on high-performing states.
- Increase inventory for top-selling product categories.
- Encourage adoption of alternative payment methods.
- Optimize freight costs to improve profitability.
- Strengthen customer acquisition strategies in lower-performing regions.
